# PrivacyGreen Datenbank

Eine öffentliche Datenbank mit Bewertungen zur **Datenschutzfreundlichkeit** und **Nachhaltigkeit** von Websites.

## Struktur
Die Daten liegen im Format `websites.json` vor und enthalten:
- `url`: Domain der Website (z. B. `www.amazon.de`).
- `privacyScore`: Bewertung von 1–5 (1 = schlecht, 5 = gut).
- `sustainabilityScore`: Bewertung von 1–5.
- `alternatives`: Ethische Alternativen.
- `sources`: Quellen der Bewertungen.

# PrivacyGreen Database

A public database with ratings on **privacy-friendliness** and **sustainability** of websites.

## Structure
Data is provided in `websites.json` format and includes:
- `url`: Website domain (e.g., `www.amazon.de`).
- `privacyScore`: Rating from 1–5 (1 = poor, 5 = excellent).
- `sustainabilityScore`: Rating from 1–5.
- `alternatives`: Ethical alternatives to the website.
- `sources`: Sources for the ratings.
